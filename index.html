พิมพ์บิล / บันทึก PDF
วิธีที่ 1 — Canon Print (แนะนำ)
1. กด "แชร์" ใน Chrome (⋮)
2. เลือก "เปิดใน Canon Print"
3. กดพิมพ์ใน Canon Print

วิธีที่ 2 — พิมพ์/PDF ตรง
1. กดปุ่ม "พิมพ์เลย" ด้านล่าง
2. Chrome จะเปิดหน้าพิมพ์
📤 แชร์รายชื่อลูกค้า
📋 ขั้นตอนการส่ง
1. กดปุ่ม "คัดลอกข้อมูล"
2. เปิด LINE → เลือกผู้รับ
3. วางข้อความแล้วส่ง
4. ให้อีกเครื่องกดแท็บ "รับข้อมูล"
ข้อมูลที่จะส่ง:
📥 ขั้นตอนการรับ
1. เปิดข้อความที่ได้รับใน LINE
2. กดค้างที่ข้อความ → คัดลอก
3. วางในช่องด้านล่าง แล้วกด "นำเข้า"
💬 แชร์ราคาผ่าน LINE
📋 ขั้นตอนการส่ง
1. กดปุ่ม "คัดลอกข้อมูล" ด้านล่าง
2. เปิด LINE → เลือกผู้รับ
3. วางข้อความแล้วส่ง
4. ให้ผู้รับกดแท็บ "รับข้อมูล" แล้ววางข้อความ
